# Changelog

## 0.1.0 (2026-07-04)

- Initial release as standalone plugin.
- Extracted from `openclaw/extensions/hierarchical/`.
- PLS (Prompt Layering System) — hierarchical prompt inheritance.
- NTS (Node-Type Tool Isolation) — tool allow/deny per node type.
- `before_prompt_build` and `subagent_spawned` hook registration.
